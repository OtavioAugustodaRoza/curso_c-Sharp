##  Objetivo

Praticar e consolidar os fundamentos da linguagem **C#**, incluindo **programação orientada a objetos**, **estruturas de controle**, **arrays**, **métodos** e muito mais.

---

## Conteúdo Estudado

- Introdução ao C# e .NET
- Estrutura básica de um programa
- Variáveis, constantes e tipos de dados
- Operadores aritméticos, lógicos e relacionais
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Métodos / Funções
- Arrays e listas
- Classes, Objetos e POO
- Exercícios práticos e projetos simples
